# Akase Divine — Student Portfolio Website

This project is a responsive multi-page website for COS 106 web technologies.

## Pages
- `index.html` — homepage with student profile, welcome message, bio, and multimedia section
- `about.html` — educational background, aspirations, technical skills, hobbies, and academic table
- `projects.html` — three sample projects with images and links
- `planner.html` — interactive academic planner with add, complete, and delete task features
- `contact.html` — validated contact form
